# 📚 Language Learning Vocabulary App
 
A modern web app for learning languages through interactive reading, vocabulary management, and AI-powered study tools.
 
## ✨ Features
 
- **Interactive Reader** - Read texts with instant word translations and save vocabulary
- **Vocabulary Manager** - Organize words by status (New, Learning, Fuzzy, Known) with notes and search
- **Flashcards** - Spaced repetition system with progress tracking
- **Word Guess Quiz** - Multiple-choice practice with filtered study sets
- **AI Tutor** - Chat with Mira for grammar help, vocabulary, and conversation practice
- **Dark Theme** - Fully responsive design optimized for mobile and desktop

- ## 🛠️ Tech Stack

**Frontend:** Next.js 14, TypeScript, Tailwind CSS, React Icons, Sonner (toasts)
**Backend:** Next.js API Routes, PostgreSQL (Vercel/Neon), NextAuth.js, bcrypt
**AI:** Google Gemini API

## Prerequisites
- Node.js 18+
- PostgreSQL database
- Google Gemini API key
  
**Deployment:** Optimized for Vercel, also supports Netlify, Railway, etc.
